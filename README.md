# gitex

🛠️ Terminal tool to prep your 🧠 codebase (whole or partial) for LLMs — clean, compress, and convert it into prompt-ready text! 🚀📦

## 📥 Installation
```bash
$ pip install gitex
```


## ▶️ Usage
```
$ gitex --help
$ gitex .             # current repository
$ gitex path/to/repo  # any repo path
$ gitex url           # repo url
...
```
